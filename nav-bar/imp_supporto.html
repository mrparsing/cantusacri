Contatti

Per qualsiasi richiesta di supporto, suggerimenti o segnalazioni di problemi, puoi creare un'issue nella GitHub Repository o inviare un messaggio compilando il form.

Domande Frequenti (FAQ)

Come posso segnalare un problema o un bug?

Se hai riscontrato un problema o un bug, puoi creare un'issue nel nostro repository GitHub oppure inviaci un messaggio attraverso il modulo di contatto nella sezione "Contatti" della pagina.

Posso suggerire un nuovo canto liturgico da aggiungere?

Sì! Se desideri suggerire un nuovo canto liturgico, puoi inviare la tua proposta tramite il modulo di contatto o direttamente nel nostro repository GitHub.

Come posso cercare un canto o un testo liturgico?

Puoi effettuare una ricerca in diversi modi:

Salmi: Utilizza la parola chiave salmo per cercare i salmi. Ad esempio, puoi scrivere salmo I domenica tempo ordinario - A.

Testo: Se conosci una parte del testo, puoi usarla per cercare un salmo o un canto. Ad esempio, cerca Signore è il mio pastore.

Messa: Per trovare canti legati alla liturgia di una messa, usa parole chiave come messa, celebrazione, domenica o solennità.

Novene: Per cercare le novene, utilizza la parola chiave novena seguita dal nome, ad esempio novena immacolata o novena di natale.

Canti generici: Se non inserisci nessuna parola chiave, la ricerca mostrerà i canti disponibili.

Se hai suggerimenti per nuovi canti liturgici, puoi inviarli tramite il modulo di contatto o direttamente nel nostro repository GitHub.
